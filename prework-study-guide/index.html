HTML

  • HTML is the foundation of a website.
  • Head elements are what coders use to edit the website.
  • Body elements are what the users see and use.
  • We can link an external CSS file to the HTML or use CSS directly inside the HTML.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is how a website is styled, can be inside HTML or be an exteral file.
  • CSS is how we can change colors, shapes, shadows the style of the a website.

Git

  • cd directory name: changes the directory you are in.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: stages the changes that has been made inside the current branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are lines off code that are like sleeper agents, they don't act unless called upon.
  • Arrays are an index of variables that can be represented as one entity.
  • The information inside of an array can selected from a funtion.
