HTML

  • HTML is the foundation of a website
  • Head elements are what coders use to edit the website
  • Body elements are what the users see and use

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • CSS is how a website is styled, can be inside HTML or be an exteral file
  • CSS is how we can change colors, shapes, shadows the style of the a website
